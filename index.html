🎯 Task Verification & Training Priority

📋 Instructions:

Rate each task on three scales (0-3). Select the data collection mode first, then expand each duty to rate its tasks. Your ratings will be saved with the DACUM chart data.

Rating Scales:
• Importance: 0=Not Important, 1=Somewhat, 2=Important, 3=Critical
• Frequency: 0=Rarely, 1=Sometimes, 2=Often, 3=Daily
• Difficulty: 0=Easy, 1=Moderate, 2=Challenging, 3=Very Difficult

📚 Learning Outcome Design Patterns

Pattern A – One-to-One: Each Performance Criterion maps to one Learning Outcome. Best for precise, assessment-driven programs.
Pattern B – Many-to-One: Multiple Performance Criteria map to one integrated Learning Outcome. Best for competency-based modules.
Pattern C – Hybrid: Mix of both patterns based on expert judgment and instructional design needs.

Quick Start: How to Use DACUM Live Pro

  1. 1
    Enter the occupation title and job title in Chart Info tab.
  2. 2
    Use AI Generation (in Duties & Tasks tab) OR add duties and tasks manually.
  3. 3
    Add knowledge, skills, and behaviors (optional).
  4. 4
    Optionally verify tasks in Task Verification tab using workshop voting or individual ratings.
  5. 5
    Competency Clusters: Group related tasks into competency clusters, define performance criteria and range.
  6. 6
    Learning Outcomes: Map performance criteria to learning outcomes for training design.
  7. 7
    Module Mapping: Organize learning outcomes into training modules.
  8. 8
    Export as PDF or Word document.

🤖 AI-Powered: Automatically generate complete DACUM drafts with duties and tasks

📝 Format for tasks: Action Verb + Task/Activity (What) + Context (Where/How/Why if relevant)

💡 Example: "Calibrate testing equipment according to manufacturer specifications"
